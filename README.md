# SUBIT‑Opera‑Atlas

**SUBIT‑Opera‑Atlas** — це перший канонічний атлас оперного світу, який класифікує ~500 найвідоміших опер за **SUBIT‑матрицею**  
**WHO × WHERE × WHEN = 64 архетипи опери**.

Проєкт поєднує структурну логіку SUBIT, музикознавчу аналітику та енергетичну топологію жанру.

---

## 🔷 SUBIT у контексті опери

SUBIT — це тривимірна система класифікації:

- **WHO** — суб’єктність (ME, WE, YOU, THEY)  
- **WHERE** — вектор дії (EAST, SOUTH, WEST, NORTH)  
- **WHEN** — фаза (SPRING, SUMMER, AUTUMN, WINTER)

Разом вони утворюють **64 типи опер**, кожен із власною драматичною, музичною та енергетичною структурою.
