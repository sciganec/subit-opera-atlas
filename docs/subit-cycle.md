# SUBIT Cycle
## Циклічна динаміка SUBIT

SUBIT‑Cycle — це мінімальна модель переходів між станами у просторі WHO × WHERE × WHEN.  
Вона описує, як система природно рухається через фази, змінюючи суб’єктність, напрямок і темпоральний стан.

Цикл складається з **чотирьох універсальних фаз**, які повторюються у будь-якому процесі:

1. **Manifestation (SPRING)** — поява, ініціація  
2. **Formation (SUMMER)** — розвиток, інтенсивність  
3. **Evolution (AUTUMN)** — трансформація, зміна  
4. **Architectonics (WINTER)** — завершення, структура, інтеграція  

Ці фази утворюють **замкнений цикл**, який може розгортатися на будь-якій суб’єктній та просторовій осі.

---

# 1. Фаза Manifestation (SPRING)
### Характер
- поява нового стану  
- ініціація дії  
- формування наміру  
- слабка структура, висока потенційність  

### Операційна логіка
Система переходить із тиші до руху.  
Визначається базова суб’єктність (WHO) та напрямок (WHERE), але ще не стабілізовані.

### Типові переходи
```
WINTER → SPRING
```

---

# 2. Фаза Formation (SUMMER)
### Характер
- інтенсивність  
- кульмінація  
- максимальна енергія  
- активна взаємодія  

### Операційна логіка
Система розгортає повний потенціал.  
Вектор WHERE стає домінантним, суб’єктність проявляється у дії.

### Типові переходи
```
SPRING → SUMMER
```

---

# 3. Фаза Evolution (AUTUMN)
### Характер
- трансформація  
- зміна форми  
- зрілість  
- переосмислення  

### Операційна логіка
Система починає змінюватися, перетворюючи накопичену енергію на нову структуру.  
Відбувається зміна або уточнення WHO.

### Типові переходи
```
SUMMER → AUTUMN
```

---

# 4. Фаза Architectonics (WINTER)
### Характер
- завершення  
- інтеграція  
- структура  
- тиша  

### Операційна логіка
Система стабілізується, формує нову архітектуру стану.  
Це фаза, де відбувається підсумок і підготовка до нового циклу.

### Типові переходи
```
AUTUMN → WINTER
```

---

# 5. Повний цикл
У мінімальному вигляді SUBIT‑Cycle виглядає так:

```
SPRING → SUMMER → AUTUMN → WINTER → SPRING → …
```

Це універсальний ритм, який працює у:

- поведінці  
- творчості  
- організаційних процесах  
- наративах  
- мистецтві  
- інтерфейсах  
- аналітичних моделях  

---

# 6. SUBIT‑Cycle у просторі WHO × WHERE × WHEN

Цикл може розгортатися:

- **всередині одного архетипу** (мікроцикл)  
- **між архетипами однієї осі** (мезоцикл)  
- **через усі три осі** (макроцикл)  

### Мікроцикл
Зміна WHEN при фіксованих WHO і WHERE.

### Мезоцикл
Зміна WHERE при фіксованому WHO.

### Макроцикл
Зміна WHO → WHERE → WHEN у повному просторі 64 станів.

---

# 7. Правила переходів

### 7.1. Природний порядок
Переходи WHEN завжди циклічні:
```
SPRING → SUMMER → AUTUMN → WINTER → SPRING
```

### 7.2. Векторна інерція
WHERE змінюється лише тоді, коли система вичерпала поточний напрямок.

### 7.3. Суб’єктна інверсія
WHO змінюється при зміні типу конфлікту або джерела дії.
