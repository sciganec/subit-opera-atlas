# Opera Archetypes
## 64 архетипів оперної драматургії у просторі SUBIT

**Opera Archetypes** — це структурна система, яка описує оперу через 64 можливі стани, утворені комбінацією WHO × WHERE × WHEN.  
Кожен архетип — це **унікальна драматургічна формула**, яка визначає:

- тип конфлікту  
- суб’єктну перспективу  
- енергетичний напрямок  
- фазу розвитку  
- психологічну температуру  

Архетипи є **універсальними**: вони не прив’язані до конкретних творів і можуть описувати будь-яку оперу.

# Висновок

**Opera Archetypes** — це універсальна типологія оперної драматургії.  
Вона дозволяє:

- класифікувати будь-яку оперу  
- аналізувати драматургічні структури  
- будувати енергетичні карти  
- створювати нові інтерфейси та освітні моделі  
- формувати SUBIT‑Opera‑64  

Це фундаментальний документ для всієї системи.
