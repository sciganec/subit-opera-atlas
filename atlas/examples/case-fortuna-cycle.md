# Case Study: *Fortuna Imperatrix Mundi*
## WE × EAST × SPRING
## Топологічний оператор циклу та енергетичний імпульс

*Fortuna Imperatrix Mundi* — це не просто вступ до *Carmina Burana*.  
Це **математичний оператор**, який запускає, структурує і замикає весь цикл твору.

Fortuna — це **архетип появи колективної сили**, а не кульмінації.

---

# 1. SUBIT‑класифікація

```
WHO   = WE
WHERE = EAST
WHEN  = SPRING
```

Юнікод‑формула:

```
WE × EAST × SPRING = 111000
```

Це **оператор імпульсу**, а не стан дії.

---

# 2. Fortuna як математичний цикл

Fortuna — це **функція**, яка діє на енергетичний простір:

```
F(x) = x + θ mod 2π
```

де  
- *x* — точка енергії  
- *θ* — кут зсуву  
- *2π* — повний цикл  

Fortuna — це **оператор повороту**, а не лінійний рух.

---

# 3. Топологія колеса

Fortuna задає **орбіту**, а не траєкторію.

```
C = { x ∈ S¹ }
```

- *S¹* — одиничне коло  
- *C* — цикл енергії  
- рух — **обертальний**, не поступальний  

Fortuna — це **колесо**, яке обертається, а не рухається вперед.

---

# 4. Енергетичний зсув

Fortuna створює **зростаючий тиск**:

```
Eₙ₊₁ = F(Eₙ)
```

Кожен оберт:

- збільшує енергію  
- ущільнює структуру  
- піднімає температуру  

Fortuna — це **цикл з наростанням**, а не рівномірний.

---

# 5. SUBIT‑топологія Fortuna

Fortuna запускає повний WE‑цикл:

```
111000 → 111100 → 111110 → 111111 → 111101 → 111000
```

Це:

- EAST → NORTH → NORTH → SOUTH → WEST → EAST  
- SPRING → SPRING → SUMMER → SUMMER → SUMMER → SPRING  

Fortuna — це **початок і кінець одночасно**.

---

# 6. Енергетична функція в структурі Carmina Burana

Fortuna виконує три ролі:

## 6.1 Імпульс (початок)
```
Fortuna_start = імпульс появи сили
```

## 6.2 Структура (рамка)
```
Fortuna = оператор циклу
```

## 6.3 Замикання (кінець)
```
Fortuna_end = повернення до початку
```

Початок = кінець → **інволюція**.

---

# 7. Музична онтологія Fortuna

## 7.1 Ритм = архетип
- ударні  
- моторика  
- повторюваність  

Ритм — це **структурний жест**, а не музичний матеріал.

## 7.2 Хор = маса
Хор не співає — він **виголошує**.

Це **голос колективу**, а не персонажів.

## 7.3 Гармонія = тиск
Гармонія не розвивається — вона **тисне**.

---

# 8. Чому Fortuna ≠ 111111

Fortuna — це **не кульмінація**, а **передвибуховий стан**.

```
111000 ≠ 111111
```

- 111000 = імпульс  
- 111111 = екстаз  

Fortuna — це **зерно**, з якого виростає весь ритуал.

---

# 9. Порівняння з іншими циклічними операторами

| Твір | Стан | Функція |
|------|-------|----------|

| *Requiem aeternam* | THEY × NORTH × WINTER | статична структура |
| *Fortuna Imperatrix Mundi* | WE × EAST × SPRING | запуск циклу |

Fortuna — найчистіший **оператор запуску**.

---

# 10. Висновок

*Fortuna Imperatrix Mundi* — це:

- оператор циклу  
- енергетичний імпульс  
- топологічний поворот

Fortuna — це **математичне серце Carmina Burana**.
