# Case Study: *Carmina Burana*
## WE × SOUTH × SUMMER
## Опера колективного екстазу та ритуальної енергії

*Carmina Burana* Карла Орфа — один із найчистіших і найпотужніших архетипів у всьому SUBIT‑просторі.  
Це не опера в традиційному сенсі, а **сценічна кантата**, яка поводиться як **опера колективної дії**.

---

# 1. SUBIT‑класифікація

```
WHO   = WE
WHERE = SOUTH
WHEN  = SUMMER
```

Юнікод‑формула:

```
WE × SOUTH × SUMMER = 111111
```

Це **максимально гарячий стан SUBIT‑простору**.

---

# 2. Чому Carmina Burana = 111111

## 2.1 WE — суб’єкт = громада
- хор — головний носій енергії  
- солісти — лише вектори хору  
- індивідуальність розчинена  

## 2.2 SOUTH — вектор = дія
- ритм домінує над мелодією  
- моторика, удари, експансія  
- музика рухається вперед, а не всередину  

## 2.3 SUMMER — фаза = кульмінація
- постійний перегрів  
- відсутність занепаду  
- відсутність рефлексії  
- емоційний пік без розрядки

# 3. Енергетична топологія твору

Carmina Burana — це **ритуальний цикл**, а не сюжет.

Юнікод‑траєкторія:

```
EAST → SOUTH → WEST → SOUTH → NORTH
```

Але домінує **SOUTH × SUMMER**.

## 3.1 Fortuna (EAST × SPRING)
Поява сили, імпульс, колесо.

## 3.2 Весна (SOUTH × SPRING)
Пробудження тілесності.

## 3.3 Таверна (SOUTH × SUMMER)
Перегрів, хаос, колективний розгул.

## 3.4 Любов (WEST × SUMMER)
Екстаз, чуттєва кульмінація.

## 3.5 Апофеоз (SOUTH × SUMMER)
Тріумф, ритуальна перемога.

## 3.6 Повернення Fortuna (NORTH × SUMMER)
Замикання циклу.

# 4. Музична онтологія

## 4.1 Ритм = суб’єкт
```
ритм > мелодія
удар > фраза
моторика > розвиток
```

## 4.2 Хор = тіло громади
- хор не коментує — він **діє**  
- хор не описує — він **є суб’єктом**  

## 4.3 Екстаз = фаза
- кульмінація без падіння  
- емоційний тиск без розрядки  
- вічне літо

# 5. Структурна логіка

Carmina Burana — це **опера моменту**, а не оповіді.

### 5.1 Відсутність психології
Немає героїв, немає внутрішніх конфліктів.

### 5.2 Відсутність розвитку
Енергія не еволюціонує — вона **пульсує**.

### 5.3 Відсутність трагедії
Немає AUTUMN, немає WINTER.

### 5.4 Циклічність
Початок = кінець.

Юнікод‑формула:

```
Fortuna_start = Fortuna_end
```
